🎓 PowerShell Verb-Noun Pattern (The Key to Everything)

  • ✅ Get-* → Retrieve information (Read-only, safe)
  • ✅ Set-* → Modify existing values
  • ✅ New-* → Create new objects
  • ✅ Remove-* → Delete objects (⚠️ dangerous)
  • ✅ Invoke-* → Execute actions
  • ✅ Export-* / Import-* → Save/load data from files
