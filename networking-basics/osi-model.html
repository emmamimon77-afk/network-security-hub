📚 The OSI Model

Open Systems Interconnection Model — A conceptual framework used to understand network communication in 7 layers.

🧠 Why Learn the OSI Model?

  • ✅ Troubleshoot networks layer by layer
  • ✅ Understand how data travels
  • ✅ Speak the language of network security

📊 The 7 Layers of OSI

Layer 7ApplicationHTTP, FTP, SMTP, DNS
Layer 6PresentationEncryption, compression (TLS, JPEG)
Layer 5SessionSession mgmt (NetBIOS, RPC)
Layer 4TransportTCP/UDP, segmentation, ports
Layer 3NetworkIP addressing, routing (Routers)
Layer 2Data LinkMAC addresses, switches, Ethernet
Layer 1PhysicalCables, fiber, voltage

📝 Data Flow (Encapsulation)

+-------------+ | Layer 7 | "Hello" (data) +-------------+ ⬇️ +-------------+ | Layer 4 | TCP segment (ports) +-------------+ ⬇️ +-------------+ | Layer 3 | IP packet (IP addresses) +-------------+ ⬇️ +-------------+ | Layer 2 | Ethernet frame (MAC) +-------------+ ⬇️ +-------------+ | Layer 1 | Bits on the wire +-------------+
