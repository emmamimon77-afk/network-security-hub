📡 Lab 3: Packet Analysis with Wireshark (Cross-Platform)

⭐⭐ Difficulty: Medium ⏱️ Time: 25-30 minutes 📖 Prereq: OSI + TCP/IP lessons

What you'll learn: Capture live packets, filter HTTP/DNS traffic, and see OSI/TCP/IP layers in real time.

Tools needed: Wireshark (free) + your browser

📋 Click for detailed instructions

Step 1: Install Wireshark

  • Linux (Ubuntu/Debian): sudo apt install wireshark
  • macOS: brew install --cask wireshark
  • Windows: Download from wireshark.org

Step 2: Start a Capture

  • Open Wireshark → Select your active network interface (Wi-Fi or Ethernet)
  • Click the shark fin icon to start capture

Step 3: Generate Traffic

  • Open browser → visit http://example.com (HTTP, not HTTPS)
  • Run: ping 8.8.8.8 in terminal
  • Run: nslookup google.com

Step 4: Apply Filters

  • Filter http → see HTTP request/response (Layer 7)
  • Filter dns → see DNS queries (Layer 7)
  • Filter icmp → see ping packets (Layer 3)
  • Click any packet → expand Ethernet (L2), IP (L3), TCP/UDP (L4)
💡 Questions to answer:
  • What's the MAC address of your default gateway?
  • Which port does HTTP use? Which source port did your browser use?
  • How many hops did your ping take?
